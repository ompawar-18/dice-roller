🎲 Dice Roller Web App

A simple and interactive Dice Roller built using HTML, CSS, and JavaScript.
Click the button to roll the dice and generate a random number between 1 and 6 with corresponding dice images.

🚀 Features
🎯 Generate random dice number (1–6)
🖼️ Dynamic dice image update
⚡ Fast and lightweight (no libraries used)
💻 Beginner-friendly project
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
dice-roller/
│── index.html
│── style.css
│── script.js
▶️ How to Run the Project
Clone the repository:
git clone https://github.com/your-username/dice-roller.git
Open the project folder
Run the app:
Double click index.html
OR
Open with Live Server (VS Code recommended)
🎮 How It Works
When user clicks "Roll Dice"
JavaScript generates a random number:
Math.floor(Math.random() * 6) + 1
Based on the number, the dice image updates dynamically
